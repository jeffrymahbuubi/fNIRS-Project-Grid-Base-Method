## Reporting Convention

All metrics reported as **Mean ± SD** across folds, with **95% CI** = mean ± 1.96·(SD/√n).  
The **Overall** row is computed from the pooled confusion matrix across all folds (equivalent to a single evaluation over all test samples).

| Symbol | Metric | Formula |
|--------|--------|---------|
| Acc | Accuracy | (TP+TN)/(TP+TN+FP+FN) |
| Sens | Sensitivity / Recall | TP/(TP+FN) |
| Spec | Specificity | TN/(TN+FP) |
| Prec | Precision / PPV | TP/(TP+FP) |
| NPV | Negative Predictive Value | TN/(TN+FN) |
| F1 | F1-Score | 2·TP/(2·TP+FP+FN) |
| BA | Balanced Accuracy | (Sens+Spec)/2 |
| κ | Cohen's Kappa | (po−pe)/(1−pe) |
| MCC | Matthews Corr. Coeff. | (TP·TN−FP·FN)/√[(TP+FP)(TP+FN)(TN+FP)(TN+FN)] |

> Positive class = Anxiety/Cognitive Load (label 1); Negative class = Control (label 0)

Key findings:
- **~15–17% accuracy drop** from 10-fold CV to LOSO, indicating moderate subject-level overfitting under fold-based CV.
- **High Sensitivity (92–95%), low Specificity (59%) across all tasks** — the model generalizes better for the anxiety/GAD class than for healthy controls. The 2:1 class imbalance (HC:GAD) may bias the model toward predicting label 0, yet Spec remains lower than Sens, suggesting the HRF patterns of misclassified HC subjects resemble the GAD cohort.
- **13/32 HC subjects (40.6%) systematically fail** across all 4 tasks with best_epoch=0, pointing to a subgroup of HC participants whose resting hemodynamic profiles overlap with the GAD task-evoked response.
- **GNG and VF produce identical LOSO confusion matrices** [[76, 52], [3, 61]], indicating near-identical per-subject classification decisions for these two tasks — a strong signal of correlated task-dependent hemodynamic representations in the HbT channel.
- **SS shows the smallest performance degradation** (Δ=−0.083) from 10-fold to LOSO, suggesting that serial subtraction produces the most subject-independent HbT patterns among the four tasks.
- **LOSO κ range: 0.428–0.459** (moderate agreement), substantially lower than 10-fold κ (0.563–0.744), confirming that fold-based CV inflates performance estimates relative to true cross-subject generalization.
